Pokemon Go dex (work-in-progress)

Abstract

As a pet project, I'm currently working on building an interactive pokedex for the mobile game Pokemon Go. The app currently displays all pokemon available in the game, their typing and the ability to check which pokemon have their shiny form available, which of course displays the shiny sprite. I also plan to add more information such as moves learned and evolution information for each pokemon.

The project is built using React on Typescript, HTML and CSS for styling and data being pulled through several APIs.
