Pokemon Draft League (work-in-progress)

Freelance project

Abstract

I was hired by the PoGo Raids Discord server as a freelance developer to create a site for their draft league. PoGo Raids hosts a draft league every three months, and my website is designed to host these tournaments, displaying member's teams and scoreboards. The website allows trainers to customize their teams and input results from their matches, although the latter is still in development.

The project is built using React on Typescript, HTML and CSS for styling on the frontend. On the backend, I built an API hosted on Vercel that handles the backend functionality and database. The database is implemented using MongoDB, hosted on Atlas.
